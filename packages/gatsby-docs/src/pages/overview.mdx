title: Overview
---

To help you with the upgrade to the new UI in IBM Social Program Management v8, you can use the SPM UI Upgrade helper, and refer to the provided guidelines to help you to manually update Social Program Management font-faces.

The SPM UI Upgrade Helper is a suite of tools that help to automate the UI upgrade process by applying Social Program Management v8 changes to a 7.0.11.0 development environment.
