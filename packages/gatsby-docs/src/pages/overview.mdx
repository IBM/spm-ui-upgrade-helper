To help you with the upgrade to the new UI in IBM Social Program Management v8, you can use the SPM UI Upgrade helper,

The SPM UI Upgrade Helper is a suite of tools that help to automate the UI upgrade process by applying Social Program Management v8 changes to a 7.0.11.0 development environment.

For detailed instructions on manually updating Social Program Management font-faces, see [Manually updating fonts](customer/customer_font_replacement).
