---
title: Coding
---

## Windows

1. Run `set UIUH_DEV=true`
2. Start the container with `dev.bat`
3. Edit the code as required
4. The container will automatically reload if running

## Linux

1. Run `export UIUH_DEV=true`
2. Start the container with `./dev.sh`
3. Edit the code as required
4. The container will automatically reload if running
